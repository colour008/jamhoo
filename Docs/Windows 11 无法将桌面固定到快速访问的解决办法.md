# Windows 11 无法将桌面固定到快速访问的解决办法

[Windows](https://wker.com/tag/windows/) 11 下，如果你发现桌面文件夹无法固定到快速访问，有一个原因，可能就是你在迁移或者固定到快速访问的时候，在其他某个地方，仍留有一个重复的桌面文件夹，找到这个文件夹，删除即可。

我有一个习惯，就是安装好 Windows 之后，会把个人文件夹下的桌面、下载、文档等文件夹迁移到另外一个硬盘上，这样重装系统或者磁盘坏了，对个人文件几乎没有任何影响，还可以把微信、QQ 等文件也放到整个下面，这样重装之后，配合 [everything](https://wker.com/powertoys-everything-plugin/) 的搜索，也可以找回原来接收的文件（微信的聊天记录好像不行）。

文件夹迁移好之后，我会把他们固定到快速访问里去，在固定到快速访问的时候，可以使用鼠标拖动，就是直接拉到快速访问，这个时候会有两个选项，一个是固定到快速访问，另一个是把文件夹重叠在另外一个文件夹上的时候会显示添加到某个文件夹。

我在固定文件夹到快速访问的过程中，把桌面文件夹不小心添加到了其他文件夹里，后来又移出来，虽然桌面正常，但是一直固定不到快速访问栏，搜索了一些网站，在[贴吧](https://tieba.baidu.com/p/8261004356)找到一个思路，说可能有重复的桌面文件夹，我按照这种思路找问题，终于发现在用户/公用 这个文件夹下发现了Desktop，里面刚好还有几个快捷方式，删掉其中一个，发现桌面上的快捷方式也被删掉了，把这个 Desktop 删掉之后，就可以正常添加了
