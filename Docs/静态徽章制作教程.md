# 静态徽章制作教程

> 打开网址：https://shields.io/

点击`Get started`，在右侧打开`Show optional parameters`（翻译后显示可选参数），然后依次填写`badgeContent`、`logo`及其他需要的选项，最后在下面选择`Execute`，就可以查看生成后效果。
