# Win10升级Win11新方法 (无TPM硬件限制、可保留文件升级)

## 一、下载镜像

微软官网下载Win11镜像：

https://www.microsoft.com/zh-cn/software-download/windows11

## 二、安装系统

* 鼠标双击打开下载的`ISO`文件，按住键盘<kbd> Shift</kbd>不松手点击鼠标右键，选择复制文件地址（路径）
* <kbd>Win</kbd>+<kbd>R</kbd>打开运行，输入复制的文件路径`"F:\setup.exe"`然后添加`/product server`，按回车键开始按照操作升级系统，可以跳过TPM硬件检测。
